Backend part :
1- npm init
2- npm install --save express express-validator mongoose
3- create src folder 
4- create index.server.js file
5- install nodemon and start server with npm start
6- for mongodb : user = root, password = root
7- npm install --save bcrypt to hash the password
8- create admin-app and install react-bootstrap
9- fil-uploader , formidable , multer, are packages used to upload files with node.js
